# Roadmap

## Round 01: idea package

- Keep product definition identical across form, PDF, and film.
- Submit the ten-slide deck and 112-second motion film.
- Name one initial public SEBI source family.
- Describe benchmark and sandbox evaluation without inventing results.

## Round 02: vertical slice

- Implement one end-to-end corpus path.
- Create one synthetic small-intermediary profile.
- Demonstrate one ambiguity routed to human review.
- Produce one Applicability Receipt.
- Demonstrate one amendment and stale-evidence event.
- Add one simulated read-only evidence connector.
- Export one Compliance Build Manifest and Sandbox Test Pack.

## Final jury

- Obtain domain-expert workflow validation.
- Report measured benchmark results.
- Demonstrate safe failure and replay reliably.
- Add a second small Corpus Pack to prove onboarding reuse.
- Present deployment, privacy, and small-intermediary economics.

## Post-competition

- Evaluate private or on-premises deployment.
- Add read-only task and evidence connectors.
- Add authority-specific SEBI, NSE, BSE, CDSL, and NSDL packs.
- Explore minimum-cohort supervisory signals only after privacy and governance review.
