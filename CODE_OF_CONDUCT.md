# Code of Conduct

## Our standard

Contributors must communicate professionally, review technical claims in good faith, respect privacy and intellectual property, and keep regulatory or financial assertions evidence-based.

Unacceptable behavior includes harassment, discriminatory language, publishing private information, deliberately introducing false claims, or misrepresenting simulated functionality as a deployed system.

## Enforcement

Report conduct concerns privately to the repository maintainers. Maintainers may edit, reject, or remove contributions and may restrict repository access when behavior threatens collaborators, project integrity, or submission credibility.

This policy applies in repository discussions, reviews, issues, and project-related communication.
